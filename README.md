# ecommerce-sales-analysis
Basic e-commerce data analysis project (junior level)

Анализ продаж интернет-магазина

О проекте

В данном проекте проведён анализ транзакционных данных интернет-магазина с целью оценки продаж и выявления ключевых бизнес-инсайтов.

---

 Цели

* Проанализировать динамику продаж
* Определить наиболее прибыльные товары
* Изучить распределение выручки по странам

---

Данные

Датасет содержит:

* InvoiceDate — дата покупки
* Description — название товара
* Quantity — количество
* UnitPrice — цена за единицу
* Country — страна покупателя

---

Используемые инструменты

* Python
* pandas
* matplotlib

---

Что было сделано

* Рассчитана выручка (Revenue)
* Проведён анализ динамики продаж по дням
* Выделены топ-10 товаров по выручке
* Проанализировано распределение продаж по странам

---

Основные выводы

* Небольшое количество товаров генерирует основную часть выручки
* Продажи сконцентрированы в одной стране
* Наблюдаются пики активности, которые могут быть связаны с акциями или сезонностью

---

Возможные улучшения

* Добавить сегментацию клиентов
* Построить дашборд (Power BI / Tableau)
* Провести когортный анализ
  
---
Бизнес инсайты

* 20% товаров генерируют основную выручку
* Продажи зависят от ключевых SKU
* Основной рынок — одна страна
